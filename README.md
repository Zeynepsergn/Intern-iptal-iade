# Iade Servis Uygulaması
-------------------------

Bu uygulama kişinin yapmış olduğu ödemeleri  iş kurallarına göre iade talebi doğrultusunda yapılan tahsilatın 
kişiye iade edilmesini ve borcun yeniden yapılandırılmasını sağlar.
